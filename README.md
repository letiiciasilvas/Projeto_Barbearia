# 💈 Fio Da Navalha

Sistema desenvolvido para a barbearia **Fio Da Navalha**, com o objetivo de oferecer uma experiência moderna e organizada para clientes e administradores.

## 📌 Sobre o Projeto

O **Fio Da Navalha** é uma aplicação voltada para o gerenciamento de uma barbearia, permitindo uma administração mais eficiente dos serviços e proporcionando praticidade no atendimento aos clientes.

O projeto foi criado com foco em simplicidade, organização e uma interface agradável, utilizando uma identidade visual moderna inspirada em tons elegantes e profissionais.

---

## ✨ Funcionalidades

- Cadastro de usuários;
- Login com autenticação;
- Agendamento de horários;
- Visualização dos serviços disponíveis;
- Gerenciamento de clientes;
- Interface intuitiva e responsiva;
- Organização e controle das informações da barbearia.

---

## 🎨 Identidade Visual

A identidade da **Fio Da Navalha** foi desenvolvida para transmitir:

- Sofisticação;
- Confiança;
- Profissionalismo;
- Tradição e modernidade.
